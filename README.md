# Test GIT
## CodeEducation
### Aula Git

* Fazer um push em um repositório qualquer no Git
* Criar um arquivo README
* Utilizar Markdown

## Configure seu diretório local

### Configurar Git no seu computador, caso ainda não tenha.

* mkdir /path/to/your/project
* cd /path/to/your/project
* git init
* git remote add origin https://ahdail@bitbucket.org/ahdail/teste.git

## Crie arquivo, faça commit e push

* Touch README.md
* git add README.md
* git commit -m "file README.md"
* push -u origin master


Agora está tudo pronto!
